# RoomTmultiferroicityInTMDs-npj2DmatAppl
Data files for publication titled "Room Temperature Multiferroicity in a Transition Metal Dichalcogenide", npj 2D Materials and Applications

Publication information:

Article.pdf is the main manuscript file.

SI.pdf is the Supplementary Information file.

Folder "Data_available_for_npj_publication" containd all experimental data files for the figures in the main manuscript and supplementary information.

Forder "SI_Files" contains the supplementary files listed and explained in the last page of Supplementary information (SI.pdf).
